Hallpass

January 07, 2017

Hallpass is a password manager focused on security via biometrics. It allows users to sign in to websites on any device by using the fingerprint sensor on their phone.

Motivation

Hallpass was conceived of at Hack-DFW 2016 as a way to consolidate password management without having to rely on a master password. It uses AES-256 encryption and supports authorizing and deauthorizing multiple devices for each user.

Stack

Backend

Node.js in combination with socket.io was used to build the api and support push/pull connections.

MongoDB was used for efficient data retrieval

Frontend

Kotlin was used to develop the native Android app Swift was used to develop the native iOS app
