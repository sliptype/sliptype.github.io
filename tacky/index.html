Tacky

September 20, 2017

Tacky is a scenario discovery and management tool. It allows you to create a hierarchical description of user interactions with a system and transform it into a list of discrete test cases.

Motivation

The linear way of writing test scenarios leaves a lot to be desired. Representing the state of a system without abstracting away the steps to get there becomes tedious and error-prone.

Stack

Vue.js was chosen because of how quickly a prototype can be built with it. There is just enough “magic” to make putting together an application a breeze.

Vuex was used as the state container. Having a centralized store makes it easy to save and share Tacky boards.

Express was used to serve the application.
